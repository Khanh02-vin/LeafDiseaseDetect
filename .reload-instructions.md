# Hướng dẫn Reload App

## Bước 1: Stop app hiện tại
Trong terminal, nhấn: **Ctrl+C**

## Bước 2: Xóa cache
```bash
# Xóa Metro bundler cache
rm -rf node_modules/.cache
rm -rf .expo

# Xóa cache trên device (optional)
rm -rf ~/Library/Developer/Xcode/DerivedData
```

## Bước 3: Restart với clear cache
```bash
bun start --clear
# hoặc
npx expo start --clear
```

## Bước 4: Rebuild trên device
- Trong app: Shake device → "Reload"
- Hoặc trong terminal: nhấn `r`

## Nếu vẫn có lỗi:
LogBox warning không ảnh hưởng chức năng - app vẫn hoạt động bình thường!
Có thể bỏ qua warning này trong development.
